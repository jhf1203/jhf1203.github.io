Skills Used | Semantic HTML, CSS, SEO Optimization

Project Summary

In this assignment I was tasked to simplify both the HTML and CSS without impacting the aesthetics or navigation of the deployed website whatsoever

Project Details

I tackled two main ideas to make this project more accessible and efficient for future updates from either myself or other collaborators: Semantic HTML: Many of the divs you see in the unsolved HTML were able to be easily converted into elements that both made sense to code alteration, as well as improved SEO visibility ("header" as an element versus a div class, for example). CSS consolidation: I identified multiple redundancies in the stylesheet that I was able to condense significantly. Multiple elements within the same container had different rules applied that were all the same, yet they were needlessly individualized as multiple classes. As a hypothetical SEO firm consistency in aesthetics is extremely important, thus by broadening class scope while minimizing class count we are able to more easily modify the page in the future and still maintain the same visual standards.
